# Finance-Forecast-Project

## Overview
This project is a personal finance simulator written in Python. It models the finances of a recent college graduate by calculating income, expenses, savings, investments, taxes, and net worth. The goal of the project was to practice Python programming while exploring real world financial concepts.

## Features
- Monthly income calculations
- Monthly expense tracking
- Net monthly cash flow
- Annual income and expense calculations
- Savings rate calculation
- 12-month sainvgs forecast
- Investment growth projections
- 5-year investment forecast
- Emergency fund recommendation
- Net worth calculation
- Salary raise scenario analysis
- Estimated tax calculations
- Debt-to-asset ratio

# Technologies Used
- Python 3

## How to Run
1. Download the project
2. Open 'main.py'
3. Run:
   '''bash
   python3 main.py
   '''
